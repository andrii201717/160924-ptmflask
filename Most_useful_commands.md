# **flask**

| Команда               | Что она делает                                | Пример команды         |
|-----------------------|-----------------------------------------------|------------------------|
| flask run	            | Запуск dev-сервера	                           | flask run              |
| flask run --port 8000 | 	С запуском на другом порту	                  | flask run --port 8000  |
| flask shell	          | Открывает интерактивный Flask Shell	          | flask shell            |
| flask routes          | 	Показывает зарегистрированные маршруты	      | flask routes           |
| flask --app <file>	   | Указать приложение (если не через .flaskenv)	 | flask --app app.py run |

# **flask db**

| Команда                           | Что она делает                                       | Пример команды                       |
|-----------------------------------|------------------------------------------------------|--------------------------------------|
| flask db init                     | 	Инициализирует папку migrations/	                   | flask db init                        |
| flask db migrate -m "<msg>"	      | Создаёт миграцию по изменениям в моделях	            | flask db migrate -m "Add User table" |
| flask db upgrade	                 | Применяет миграции к базе                            | 	flask db upgrade                    |
| flask db downgrade                | 	Откатывает последнюю миграцию                       | 	flask db downgrade                  |
| flask db downgrade <migration_id> | 	Откат до конкретной миграции	                       | flask db downgrade 3f2a1d0c5a1b      |
| flask db history                  | 	Показывает историю миграций	                        | flask db history                     |
| flask db current	                 | Показывает текущую миграцию	                         | flask db current                     |
| flask db stamp <migration_id>	    | Отмечает миграцию как применённую (БЕЗ запуска SQL)	 | flask db stamp head                  |

# **GIT**

> **Раота с ветками**

| Команда                                | Что делает                    |
|----------------------------------------|-------------------------------|
| git checkout -b <branch_name>          | Создать ветку и сразу перейти |
| git checkout <branch_name>             | Переключиться на другую ветку |
| git branch -d <branch_name>            | Удалить локальную ветку       |
| git push origin --delete <branch_name> | Удалить удалённую ветку       |
---
> **Отслеживание изменений**

| Команда                         | Что делает                      |
|---------------------------------|---------------------------------|
| git status                      | Показать изменённые файлы	      |
| git diff                        | Показать дифф по изменениям	    |
| git log --oneline --graph --all | Показать коммиты текущей ветки	 |

---
> **Стейдж и коммиты изменений**

| Команда                       | Что делает                 |
|-------------------------------|----------------------------|
| git add .                     | Добавить всё	              |
| git add file.py               | Добавить конкретный файл	  |
| git restore --staged file.py  | Убрать файл из стейджа	    |
| git commit -m "<your_commit>" | Создать коммит	            |
| git commit --amend            | Изменить последний коммит	 |

---

> **Слияние историй и изменение истории кммитов**

| Команда                                                | Что делает            |
|--------------------------------------------------------|-----------------------|
| git merge <branch_name>                                | Слить ветку	          |
| git rebase <branch_name>                               | Перебазировать ветку	 |
| git rebase -i HEAD~3 → затем в редакторе pick → squash | Сквошнуть коммиты	    |
| git cherry-pick <commit_hash>                          | Cherry-pick коммита	  |

---
> **Push / Pull**

| Команда                       | Что делает          |
|-------------------------------|---------------------|
| git push origin <branch_name> | Отправить ветку	    |
| git pull origin <branch_name> | Забрать обновления	 |

---
> **Стэш — временное сохранение изменений (заморозка)**

| Команда        | Что делает           |
|----------------|----------------------|
| git stash      | Сохранить изменения	 |
| git stash list | Список стэшей	       |
| git stash pop  | Вернуть последний	   |
| git stash drop | Удалить стэш	        |
